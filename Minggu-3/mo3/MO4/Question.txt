Apa yang di maksud dengan CSS, CSS adalah Cascading Style Sheets yaitu sebuah bahsa pemrograman yang di gunakan untuk mendefenisikan tampilan atau styling pada halaman web

Manfaat dari penggunaan CSS adalah memisahkan antara struktur dan presentasi halaman web, sehingga mempermudah pengelola dan pemeliharaan halaman web

aturan yang ada pada CSS
